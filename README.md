# ArduinoJumper
Проект представляет собой видеоигру на ардуино.
Он состоит из LCD I2C дисплея, кнопки, ардуино UNO, питания, зуммера и выключателя.
В этой игре вам нужно перепрыгивать через деревья и не врезаться в птиц.
